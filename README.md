# lightmetal

A Clojure server in the backend and ClojureScript in the frontend. Used for private only, not yet tested.

Implemented:
- reagent √
- figwheel √
- ring as backend server X

## Usage

Development mode
```
lein figwheel
```
Production mode (not yet working well)
```
lein cljsbuild once prod
```
